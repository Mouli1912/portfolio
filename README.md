## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

### Navigate to the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```
